# Gestor de Videojuegos
Esta web es un Gestor de videojuegos donde puedes crear una cuenta para tener una lista de videojuegos donde puedes añadir y eliminar los que quieras, puedes marcarlos como completados o como jugando y ponerles una nota a cada uno 1-10 que afectará a su nota madia calculada con la puntuacion que le han dado otros usuarios.

Tambien puedes añadir amigos enviando solicitudes de amistad o acepatando las que te envien para poder ver vuestras listas de videojuegos mutuamente.
